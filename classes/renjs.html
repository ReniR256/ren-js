Class RenJS

This is the main exported class.

import RenJS from "@renproject/ren";

It's initialized with a network, which controls both the RenVM network and Ethereum chain to use:

new RenJS(); // Same as `new RenJS("mainnet");`
new RenJS("testnet");
new RenJS({ ...NetworkMainnet, lightnodeURL: "custom lightnode URL" });

A second optional parameter lets you provide a RenVM RPC provider or a lightnode URL.

It then exposes two main functions: lockAndMint and burnAndRelease.
